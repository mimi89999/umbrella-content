title: Les risques des appareils portables
---
Si vous avez un smartphone, un ordinateur portable ou une tablette, vous transportez probablement une quantité énorme de données avec vous à tout moment :

*    Contacts sociaux ;
*  Mots de passe et détails du compte ;
*  Informations financières et médicales ;
*  Communications privées ;
*  Documents personnels et photos ;
*  Informations confidentielles sur les autres.

Votre appareil peut vous être volé relativement facilement :

*    Saisi à la frontière ;
*  Volé dans la rue ;
*  Cambriolé de votre maison.

Si quelqu’un qui utilise votre appareil peut accéder à vos données, il peut le copier en quelques secondes.

Dans cette leçon, découvrez comment rendre plus difficile l’accès de vos données à un voleur de votre appareil.

Apprenez comment chiffrer les fichiers en utilisant Veracrypt dans [Protection des Fichiers Avancé](umbrella://information/protecting-files/advanced).
